# Security Policy

## Supported Versions

The following versions of this project are currently being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please report it to us responsibly.

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please send an email to [INSERT EMAIL ADDRESS].

After your report is received, we will:

1. Acknowledge receipt of the report within 48 hours.
2. Provide an estimated timeframe for a fix.
3. Notify you once the vulnerability has been resolved.

We appreciate your help in keeping this project secure.
